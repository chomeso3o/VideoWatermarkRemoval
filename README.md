一、基础教程

可加V代搭建

2、微信开发者工具，导入此文件夹，输入自己的appid


3、打开文件pages/index/index.js，去接口网站https://api.wxshares.com注册账号，获取自己的KEY替换上

最后上传完成，自己去小程序后台提审版本即可，审核通过后就能上线了
